Geodex Search


Geodex.org is an implementation of Project 418. That is, it is simply one way in this the web architecture patterns used in P418 could be used to develop an end user experience.

Other Interfaces

  • Additional P418 Search UI (opens in new window)
  • Spatial Search Example (opens in new window)
  • Notebook repository. for some R and Jupyter examples.

All code for this and other search interfaces is hosted on GitHub for people to inspect. They leverage the GeoDex APIs to perform their searches.

The real stars of P418 are unquestionably the community of data providers. See the About section for a list of link to their sites.

Project 418 is funded by the National Science Foundation through the EarthCube Science Support Office (ESSO)
