In the process of indexing a set of data sets are generated that may be of interest to others. Those files are listed and made available here.

Data file types available:

  • The spatial index is a geohash based one. The dataset here is an extraction of that to GeoJSON.
  • The RDF graphs are the exact graphs loaded into the P418 triplestore
  • The free text index is not available. This is due to the fact it is rather larger and also uses a binary format readable by the Go Lang based Bleve package. If, given this, a copy is desired, you can get in touch with ESSO.
  • The provence graph that is a rather simple provenance example that only records the actions of the gleaner package.
  • The query log graph is generated by the P418 web services. It is a log of requests and responses.
